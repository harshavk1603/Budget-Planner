# Documentation Assets

## Screenshots

Add app screenshots to `docs/screenshots/` following this naming convention:

- `login.png` — Login/sign-up page
- `dashboard.png` — Main dashboard with stat cards, income sources, savings goals, recent transactions
- `expenses.png` — Expense tracker with search, filters, and budget limits panel
- `reports.png` — Reports page with pie chart, bar chart, and monthly summary

Once screenshots are added, update the Screenshots section in `README.md` to display them.
